## 安装

```sh
$ npm install @rc5/nut
```

## 功能特性

- 💄 使用 TypeScript 编写，这意味着自定义类型不会有问题
- 🫂 可用于客户端（Web 浏览器）和服务器（例如 node.js）上的 JavaScript 应用程序
- 🍇 包含丰富的处理不同方面问题的函数。（不断更新中！）
- 🥊 高测试覆盖率
- 🧩 支持构建更小的产物

## 使用

```js
// 在浏览器中使用
import { isBoolean } from "@rc5/nut";

// 在 node 中使用
import { isBoolean } from "@rc5/nut/node";
```
